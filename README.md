# the_datascience_guy

Personal portfolio of a data scientist & ML engineer — case studies, ML demos, writing, and open-source work.

Built with Vite, React, TypeScript, Tailwind CSS, and shadcn/ui. Maintained via [Lovable](https://lovable.dev/projects/8e80c706-863c-448b-94e9-08dda9933e5d).

## Branches

- `main` — active development
- `vision` — frozen snapshot of the original design vision, kept as a visual reference

## Local development

```sh
npm i
npm run dev
```

## Deployment

Open the [Lovable project](https://lovable.dev/projects/8e80c706-863c-448b-94e9-08dda9933e5d) and click **Share → Publish**, or connect a custom domain via Project → Settings → Domains.
